# Крестики‑нолики (Minecraft‑стиль)


Простая игра на Python с графическим интерфейсом в стиле Minecraft.

## Как запустить
1. Установите Python 3.8+
2. Скачайте репозиторий: `git clone https://github.com/Surene0/tic-tac-toe-minecraft.git`
3. Запустите: `python tic_tac_toe.py`

## Особенности
- Интерфейс в пиксельном стиле Minecraft
- Подсчёт очков
- Кнопка сброса игры
